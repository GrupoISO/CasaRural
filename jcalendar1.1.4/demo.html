To run the JCalendar demonstration applet in your browser, the Java Plug-in has to be installed. This applet demonstrates the JCalendar Java bean, a Java date chooser composed of several other Java beans (a JDayChooser, a JMonthChooser and a JYearChooser). All these beans have a locale property. If you change the locale with the locale chooser, the names of the months and weekdays change (make sure, that you have bound the Java Plug-in to an international JRE, otherwise you might see only one locale!). Just for demonstration, the chosen date is displayed below the locale chooser. Since I think it looks just great, I have packaged JCalendar with the free Kunststoff look & feel (made by Incors).
